#### Setting up Virtual Env

`pyenv virtualenv 3.6.2 django_app`
`pyenv activate django_app`

#### Downloading Django
If you don't have pip, install pip.
Then,
`pip install -r requirements.txt`
to install all the dependencies for this project.
